# 青砚 Design System — 东方精密工作台

> 融合 Linear 的设计精度 × 中式文房美学，为 AI 工作助理打造的设计规范。

## 1. 视觉哲学

青砚的设计建立在"宣纸为底、墨色为骨"的理念上 — 一个温润的浅色画布（`#f2f0eb`），内容如墨迹般从纸面浮现。所有 UI 元素存在于精心校准的色彩层级中：从几乎不可见的边框（`rgba(26,36,32,0.05)`）到沉稳的墨色文字（`#1a2420`）。这不是简单的浅色主题 — 宣纸是原生介质，信息密度通过黑色透明度的微妙梯度来管理。

字体系统基于 Inter，启用 OpenType 特性 `"cv01"` 和 `"ss03"`，赋予字体更几何、更精密的气质。采用三级权重系统：400（阅读）、500（UI 强调）、600（标题宣告）。大号标题使用紧凑的负字间距，创造出"工程精密"而非"设计花哨"的感觉。中文回退至 PingFang SC / Microsoft YaHei / Noto Sans SC。

色彩系统以黛青（`#2b6055`）为唯一品牌色，其余全部为暖灰色阶。黛青仅用于 CTA、激活态和品牌标记。温铜金（`#b08d5b`）极度克制使用，仅出现在品牌装饰场景。

**核心特征：**
- 宣纸浅底：`#f2f0eb` 页面背景，`#faf8f4` 卡片表面，`#ffffff` 最高层
- 墨色深侧栏：渐变 `#0f1a1c → #142224 → #111d1f`
- Inter + `"cv01", "ss03"` 全局启用
- 三级权重：400（正文）、500（UI/导航）、600（标题/强调）
- 品牌黛青：`#2b6055`（背景）/ `#224e45`（悬停）— 系统中唯一色彩色
- 超薄半透明边框：`rgba(26,36,32,0.05)` ～ `rgba(26,36,32,0.09)`
- 多层阴影系统模拟"墨晕"效果
- 8px 基础间距网格

## 2. 色板与角色

### 背景层级
- **宣纸**（`#f2f0eb`）：页面背景 — 温暖的米白，带微不可察的绿灰色调
- **玉白**（`#faf8f4`）：卡片、面板表面 — 比宣纸亮一级
- **纯白**（`#ffffff`）：最高层 — 弹窗、下拉、聚焦面板
- **浅灰面**（`#f5f3ef`）：hover 态、轻微凸起组件

### 文字与内容
- **墨色**（`#1a2420`）：主文字 — 非纯黑，带微暖绿调，在浅底上不刺眼
- **次墨**（`#374843`）：正文、描述 — 深墨绿灰
- **松烟灰**（`#6e7d76`）：占位符、元数据、弱化内容
- **淡墨**（`#9ca8a2`）：时间戳、禁用态、最弱标签

### 品牌色 · 黛青
- **黛青**（`#2b6055`）：主品牌色 — CTA 按钮、品牌标记、关键交互
- **深黛**（`#224e45`）：悬停态 — 更深沉的黛青
- **黛青柔光**（`rgba(43,96,85,0.09)`）：选中态背景、轻着色
- **黛青辉光**（`rgba(80,160,140,0.22)`）：激活态发光效果

### 语义色
- **翠**（`#2e7a56`）：成功 / 完成 — 背景 `#eef6f1`
- **铜**（`#9a6a2f`）：警告 / 注意 — 背景 `#faf3e7`
- **朱砂**（`#a63d3d`）：危险 / 错误 — 背景 `#faf0ef`
- **青瓷**（`#2d6a7a`）：信息 / 中性提示 — 背景 `#ebf2f5`

### 边框与分隔
- **边框默认**（`rgba(26,36,32,0.09)`）：标准卡片、输入框
- **边框轻微**（`rgba(26,36,32,0.05)`）：最细微的分隔
- **侧栏边框**（`rgba(255,255,255,0.08)`）：暗色侧栏内部
- **卡片边框**（`rgba(26,36,32,0.05)`）：卡片间隔

### 侧栏（暗色系统）
- **侧栏背景**：渐变 `#0f1a1c → #142224 → #111d1f`
- **侧栏文字**（`#d0d8d2`）：主导航文字
- **侧栏弱化**（`rgba(208,216,210,0.42)`）：分组标题
- **侧栏悬停**（`rgba(255,255,255,0.05)`）
- **侧栏激活**（`rgba(43,96,85,0.28)`）：当前页面

### 遮罩
- **模态遮罩**（`rgba(0,0,0,0.5)`）：对话框背景 — 配合 `backdrop-blur(8px)`

### 装饰（极度克制）
- **温铜金**（`#b08d5b`）：品牌 Logo 渐变、特殊徽章 — 仅装饰用途
- **金柔光**（`rgba(176,141,91,0.10)`）：金色微着色

## 3. 字体规范

### 字体族
- **主字体**：`Inter`，回退：`"PingFang SC", "Microsoft YaHei", "Noto Sans SC", system-ui, sans-serif`
- **等宽字体**：`"SF Mono", "Fira Code", ui-monospace, Menlo, monospace`
- **OpenType 特性**：`"cv01", "ss03"` 全局启用

### 层级

| 角色 | 大小 | 权重 | 行高 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| Display | 32px (2rem) | 600 | 1.15 | -0.7px | 页面大标题 |
| Heading 1 | 24px (1.5rem) | 600 | 1.25 | -0.4px | 区块标题 |
| Heading 2 | 20px (1.25rem) | 600 | 1.3 | -0.3px | 卡片标题 |
| Heading 3 | 16px (1rem) | 600 | 1.4 | -0.1px | 子标题 |
| Body Large | 16px (1rem) | 400 | 1.6 | normal | 正文（大） |
| Body | 14px (0.875rem) | 400 | 1.6 | normal | 默认正文 |
| Body Medium | 14px (0.875rem) | 500 | 1.5 | normal | 导航、标签 |
| Small | 13px (0.813rem) | 400 | 1.5 | -0.08px | 辅助文字 |
| Small Medium | 13px (0.813rem) | 500 | 1.5 | -0.08px | 强调辅助文字 |
| Caption | 12px (0.75rem) | 500 | 1.4 | normal | 元数据、小标签 |
| Micro | 11px (0.688rem) | 500 | 1.4 | 0.02em | 徽章、最小标签 |
| Tiny | 10px (0.625rem) | 500 | 1.4 | 0.05em | 组标题（常大写） |
| Mono | 13px (0.813rem) | 400 | 1.5 | normal | 代码 |

### 原则
- **500 是 UI 签名权重**：介于 regular 和 medium 之间，温和强调但不粗犷
- **600 用于标题**：所有标题/宣告性文字用 600，最重不超过 700
- **大号紧凑**：24px+ 标题使用负字间距，越大越紧凑
- **OpenType 是身份**：`"cv01", "ss03"` 让 Inter 从通用字体变为品牌字体

## 4. 组件样式

### 按钮

**主按钮（Primary）**
- 背景：`#1a2420`（墨色）
- 文字：`#ffffff`
- 内边距：8px 16px
- 圆角：8px
- 悬停：`#1a2420/90`
- 用途：主要操作

**品牌按钮（Brand CTA）**
- 背景：`#2b6055`（黛青）
- 文字：`#ffffff`
- 内边距：8px 16px
- 圆角：8px
- 悬停：`#224e45`
- 用途：AI 相关、品牌级 CTA

**次要按钮（Secondary/Ghost）**
- 背景：`rgba(255,255,255,0.8)` / `white/80`
- 文字：`#1a2420`
- 边框：`1px solid rgba(26,36,32,0.09)`
- 圆角：8px
- 悬停：`#ffffff`
- 用途：次要操作

**小工具按钮（Toolbar）**
- 背景：`rgba(26,36,32,0.04)`
- 文字：`#6e7d76`
- 圆角：6px
- 字体：12px weight 500
- 用途：工具栏快捷操作

### 卡片与容器
- 背景：`#faf8f4`（玉白）— 半透明时用 `rgba(250,248,244,0.7)`
- 边框：`1px solid rgba(26,36,32,0.05)`（默认）或 `rgba(26,36,32,0.09)`（强调）
- 圆角：12px（标准卡片）、16px（大面板）、8px（紧凑卡片）
- 阴影：`0 1px 0 rgba(26,36,32,0.03), 0 6px 24px rgba(26,36,32,0.05)`
- 悬停：`border-color` 加深到 `rgba(26,36,32,0.12)` + 轻微阴影增强

### 输入框
- 背景：`#ffffff` 或 `rgba(255,255,255,0.8)`
- 文字：`#1a2420`
- 占位符：`#9ca8a2`
- 边框：`1px solid rgba(26,36,32,0.09)`
- 圆角：8px
- 聚焦：`ring-2 ring-accent/20`
- 内边距：10px 12px
- 最小高度：44px（触控友好）

### 徽章与标签

**状态徽章**
- 成功：背景 `#eef6f1`，文字 `#2e7a56`
- 警告：背景 `#faf3e7`，文字 `#9a6a2f`
- 危险：背景 `#faf0ef`，文字 `#a63d3d`
- 信息：背景 `#ebf2f5`，文字 `#2d6a7a`
- 圆角：9999px
- 字体：11px weight 500

**品牌标签（Beta/MVP）**
- 背景：`rgba(43,96,85,0.15)`
- 边框：`rgba(80,160,140,0.2)`
- 文字：`emerald-300/70`
- 字体：9px weight 500

## 5. 布局原则

### 间距系统（8px 网格）
- 基础单位：8px
- 核心节奏：4px、8px、12px、16px、20px、24px、32px、40px、48px
- 微调值：2px、6px（用于光学对齐）

### 容器
- 最大内容宽度：`max-w-7xl`（1280px）
- 页面内边距：`px-4 py-4 md:px-6 md:py-5`
- 卡片内边距：`p-4 md:p-5`
- 区块间距：`space-y-5`（20px）

### 留白哲学
- **宣纸即留白**：温润的背景色本身就是呼吸空间，内容从中浮现
- **紧凑标题，宽松环绕**：标题用负字间距压缩，但周围保持慷慨的空间
- **区块隔离**：区块间通过 20px 间距分离，边框仅在需要时出现

### 圆角系统
- 微型（4px）：行内徽章、工具栏按钮
- 标准（6px）：小容器、列表项
- 舒适（8px）：按钮、输入框
- 卡片（12px）：卡片、下拉、弹出层
- 面板（16px）：大面板、区块容器
- 大号（20px）：认证页面板
- 全圆（9999px）：标签、筛选器
- 圆形（50%）：头像、状态点

## 6. 层深与高度

| 层级 | 处理方式 | 用途 |
|------|---------|------|
| 平面 L0 | 无阴影，`#f2f0eb` 背景 | 页面背景 |
| 微浮 L1 | `0 1px 0 rgba(26,36,32,0.03)` | 工具栏按钮 |
| 表面 L2 | `#faf8f4` 背景 + `1px solid rgba(26,36,32,0.05)` | 卡片、输入框 |
| 卡片 L3 | `0 1px 0 rgba(26,36,32,0.03), 0 6px 24px rgba(26,36,32,0.05)` | 内容卡片 |
| 浮动 L4 | `0 4px 6px rgba(26,36,32,0.03), 0 20px 40px rgba(26,36,32,0.08)` | 下拉、弹出 |
| 对话 L5 | L4 + `backdrop-blur(8px)` | 模态、命令面板 |
| 聚焦 | `ring-2 ring-accent/20 outline-none` | 键盘焦点 |

## 7. 规范守则

### 必须
- 所有 Inter 文字启用 `font-feature-settings: "cv01", "ss03"`
- 用 500 作为 UI 默认强调权重
- 24px+ 标题使用负字间距
- 背景用 `#f2f0eb`，卡片用 `#faf8f4`
- 边框用半透明 `rgba(26,36,32,0.05-0.09)`
- 品牌黛青仅用于交互/CTA 元素
- 卡片悬停时边框加深 + 轻微阴影增强
- 所有可交互元素最小高度 44px（iPad 触控）
- 间距遵循 8px 网格

### 禁止
- 禁止在页面主体用纯白 `#ffffff`（用 `#f2f0eb` 或 `#faf8f4`）
- 禁止在 UI 装饰性使用黛青色（仅交互/CTA）
- 禁止使用 700+ 权重（最重 600）
- 禁止用不透明实色边框（边框必须半透明）
- 禁止在浅色背景上用深色重阴影（保持墨晕轻盈感）
- 禁止跳过 OpenType 特性（缺少 cv01/ss03 就不是品牌字体）
- 禁止在侧栏引入暖色（侧栏严格冷色墨绿系）

## 8. 响应式

### 断点
| 名称 | 宽度 | 变化 |
|------|------|------|
| 手机 | <640px | 单列，紧凑内边距 |
| iPad 竖屏 | 640-768px | 两列网格开始 |
| iPad 横屏 | 768-1024px | 完整卡片网格 |
| 桌面 | 1024-1280px | 标准桌面，完整导航 |
| 大屏 | >1280px | 完整布局 |

### iPad 适配（核心设备）
- 所有输入框 `font-size: 16px`（防止 iOS 缩放）
- 按钮/输入最小高度 44px
- 支持 Apple Pencil Scribble（手写转文字）
- PWA 全屏 + safe-area 支持
